Estrutura Html.

voltar para cima

Nessas aulas aprendemos uma estrutura de html onde iremos criar nossa página. Essa estrutura consiste em 3 partes: html, head e body . Cada parte tem uma função importante dentro da nossa página e iremos aprender melhor sobre cada uma.

  • HTML

Para o nosso código html precisamos claramente do elemento < html > < /html> , tudo da página ficará dentro desse código para que o programa entenda que o que estamos trabalhando é através dessa linguagem.

  • HEAD

Bom todo código tem seu começo ou seu cabeçalho e basicamente é isso que define essa parte do código, aqui você dará o nome da sua página ou guia e também com alguns ajustes poderá até mesmo colocar compatibilidade entre outras informações para carregar a página.

  • BODY

Aqui é onde toda a "mágica" acontece, todos os elementos tags e textos colocados aqui serão introduzidos na sua página, é onde você faz toda a funcionalidade da sua página e lhe trará mais aprendizado conforme for praticando.

O Que São Tags?

voltar para cima

Tags são funcionalidades dentro do html e outras linguagens de programação. As tags são usadas para diversas coisas, desde sublinhar textos ou aumentá-los até inserir imagens, gifs ou bitmaps na sua página.

Existem diversas tags e nesse site da W3SCHOOLS poderá encontrar muitas, senão todas elas, mas aqui iremos aprender somente algumas, pois ensinaremos o que temos conhecimento até o momento.

A tag H é a tag de títulos e subtítulos começando no H1 onde é o principal até o H6 que serve para um subtítulo dentro de outro e dentro de outro. Enfim essa tag vem com um padrão quando inserida porém temos algumas tags de edição que podem até certo ponto ajudar editar nossas página, mas iremos ver isso mais a frente.

Edições de Textos

voltar para cima

Essa parte é uma das que mais gostei, é o que incrementa tudo que está acima, aqui veremos uma pequena lista das tags básicas.

  • negrito < strong>
  • itálico < i>
  • sublinhado < u>

    • mas também temos tags para aumentar e diminuir fontes exemplo:

  • < font size="numero" > Define o tamanho da fonte.
  • < font size="+numero" > Aumenta a fonte.
  • < font size="-numero" > Diminui a fonte<./li>
  • < small > < /small > torna uma parte da font menor.
  • Tag de marcação < mark > < /mark >, essa simplesmente marca um texto
  • Tag de fonte < font face="verdana" > Define o nome da fonte.
  • < font color="color-name" > Define a cor da fonte
  • < font color="color-value > Define a cor através do valor direto.

Links, imagens e listas

voltar para cima

Bem, vocês viram como temos muitas funcionalidades dentro de pouca coisa, agora pra finalizar irei falar um pouco sobre links e listas ordenadas e não ordenadas.

  • Links

Esses tópicos finais são muito importante, para deixar tudo organizado e funcional dentro de uma página. A tag pra inserir link é < a> < /a> dentro dessa tag também iremos usar href= "link". Nesse formato podemos juntar todas as nossas páginas e organizar nosso trabalho, assim como inserir links externos que leve o usuário para algum outro site.

Todos os botões funcionais irão precisar dessa tag então usem e nunca esqueçam de linkar suas páginas sempre que fizer o botão.

  • Imagens

Imagens dentro de um site é algo super comum, então com a tag < img scr= "link"/> poderemos inserir uma imagem em nosso site, essa imagem pode ser de uma pasta ou de um outro site basta colar o link que automaticamente irá migrar para a página e com a tag < width="numero"> poderemos aumentar ou diminuir a imagem.

  • Listas Ordenadas e Não Ordenadas

  • Listas ordenadas são as listas númericas, onde necessariamente seguem a ordem 1,2,3... As listas ordenadas possuem a tag < ol> < /ol>, mas essa lista precisa ser acompanhada da tag < li> < /li> para que haja essa ordem e possa escrever os tópicos.
  • Listas não ordenadas, são aquelas que normalmente ou possuem uma bolinha de tópico ou mesmo um hífen, depende de como irá modificar mais a frente, mas não é uma lista que siga uma ordem necessariamente.Essas listas usam a tag < ul> < /ul>, mas a construção dela também necessita da tag acima para que funcione.

Nos sugeriram usar outras tags então preparei esse extra pra apresentar mais 2(duas)tags.

  • Del

Uma tag que deleta um texto e insere um novo texto em seu lugar. Vem inserida da tag < ins> < /ins > para colocar o novo texto no lugar.

Fonte: stackoverflow.com.br

  • ABBR

< abbr title="nome"> < /abbr> para que possa abreviar uma palavra e quando passado em cima ver o nome completo dela exemplo: Pedro I como viram o nome ta abreviado, porém utilizei o código para ver o nome completo dele, essa é uma tag muito boa pra visualização e ajuda muitos usuários.

Fonte: ehgomes.com.br
